🪟 First-Time Setup (Windows)

1

Download AltServer for Windows

Go to altstore.io → Download → Windows
→ altstore.io
2

Install AltServer & iTunes

Run the AltServer installer. It will also prompt you to install iTunes from Apple's website (not the Microsoft Store version) — this is required.
3

Plug your iPhone into the PC

Use a USB cable. Trust the computer when your iPhone asks.
4

Install AltStore on your iPhone

Click the AltServer icon in the Windows system tray → "Install AltStore" → pick your iPhone. Sign in with your Apple ID when prompted.
5

Trust the app on your iPhone

Go to iPhone Settings → General → VPN & Device Management → tap your Apple ID → Trust.
6

Open AltStore → Sources tab → tap +

Paste this URL:
https://osmandeol.github.io/portfolio-tracker-ios/altstore-source.json
7

Find Portfolio Tracker → tap Free → Install

Done! The app is now on your iPhone. AltStore will automatically re-sign it every 7 days in the background when your phone is on home Wi-Fi.

🔄 After Setup

✓

Leave AltServer running in the system tray

It uses almost no resources. Your app silently re-signs every 7 days on home Wi-Fi — you never touch it.
✓

Updates happen automatically

When a new version is released you'll see a badge in AltStore. Tap Update — same as the App Store.
✓

Set your server URL

Open the app → Settings tab → enter your Flask server's IP and port (e.g. http://192.168.1.x:5050). Tap "Test Connection" to verify.
